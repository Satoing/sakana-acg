# Sakana动漫

> 使用vue + electron制作的动漫客户端。

**兴趣使然的一个项目，数据源为Age动漫和Bangumi以及必应图片。web的前端和后端都部署在vercel上，完全是0成本的，所以不会有任何广告。**
